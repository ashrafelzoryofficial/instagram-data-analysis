# Instagram Data Analysis

This project analyzes Instagram post data using Python.

## Tools

* Python
* Pandas
* Matplotlib
* Seaborn

## Project Steps

1. Load the dataset
2. Clean the data
3. Explore the data
4. Analyze relationships between variables
5. Create visualizations

## Questions

* What is the average number of impressions?
* Do more likes increase impressions?
* Do more likes increase follows?
* Do hashtag impressions increase total impressions?
* Do profile visits increase follows?

## Insights

* Posts with more likes usually have more impressions.
* Hashtags help posts reach more people.
* Profile visits can lead to more followers.
